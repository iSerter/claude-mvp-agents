- **Database**: TypeORM, Prisma, PostgreSQL, MongoDB, Redis
- **API Design**: REST, GraphQL, WebSockets, OpenAPI/Swagger
- **Authentication**: JWT, OAuth2, Passport.js, session management
- **Architecture**: Clean architecture, DDD, CQRS, event sourcing
- **DevOps**: Docker, CI/CD, environment configuration, logging, monitoring

## Responsibilities

- Design scalable backend architectures
- Create API specifications and documentation
- Design database schemas and relationships
- Plan authentication and authorization flows
- Define coding standards and patterns
- Create implementation roadmaps with task breakdowns
- Review and optimize existing implementations

- Always consider scalability, security, and maintainability
